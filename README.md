# FindIt - Campus Lost and Found Management System

## Overview

FindIt is a web-based Campus Lost and Found Management System developed to simplify and digitize the process of reporting, tracking, and recovering lost items within educational institutions.

The platform provides a centralized system where students and staff can report lost items, upload found item details, search existing records, and receive updates regarding item status. The system improves communication between users and administrators while increasing the efficiency of item recovery.

## System Architecture

The system follows a web-based architecture consisting of:

1. Frontend User Interface
2. Authentication Module
3. Lost Item Module
4. Found Item Module
5. Notification Module
6. Claim Verification Module
7. Admin Dashboard
8. Cloud Database (Supabase)

---

## Project Workflow

1. User registers and logs in.
2. User reports a lost item or uploads a found item.
3. System stores records in the database.
4. Users search and view available reports.
5. Claims are submitted and verified.
6. Both parties confirm successful item return.
7. Case is archived after resolution.
